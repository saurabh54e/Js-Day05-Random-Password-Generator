🚀 Day 5/30 – Random Password Generator

Continuing my 30 Days JavaScript Challenge, I built a modern Password Generator Web App using Vanilla JavaScript.

## Tech Used:
• HTML
• CSS
• Vanilla JavaScript

## Features:
• Generate strong random passwords
• Adjustable password length slider
• Includes uppercase, lowercase, numbers & symbols
• Fisher-Yates shuffle for better randomness
• One-click copy to clipboard
• Fully responsive design
• Smooth UI animations

## What I Learned:
• DOM manipulation & event handling
• Randomization logic in JavaScript
• Clipboard API (async/await)
• Writing reusable helper functions
• Responsive UI design with Flexbox & media queries

## Next Improvements:
• Password strength meter
• Show / hide password toggle
• Toast notifications
• Save password history
